# Productivity Tools

This repository collects various tools, scripts, and configurations designed to boost productivity.

## Structure

- `scripts/`: Contains executable scripts (Bash, Python, Ruby, etc.) for automation.
- `docs/`: Documentation and guides.

## Usage

### Prerequisites

- Go 1.23+

### Running the CLI

```bash
go run main.go
```

### Adding new commands

```bash
cobra-cli add [command name]
```
